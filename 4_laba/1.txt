1. Цель лабораторной работы: познакомиться с основными возможностями
использования языка SQL для манипулирования данными.
2. Задание к лабораторной работе:
2.1 Выполнить запрос на выборку нескольких столбцов одной таблицы базы
данных;
2.2 Выполнить запрос на выборку значений нескольких столбцов одной
таблицы базы данных, удовлетворяющих одному условию ;
2.3 Выполнить запрос на выборку значений нескольких столбцов одной
таблицы базы данных, удовлетворяющих нескольким условиям (AND, OR,
BETWEEN... AND...) с исключением строк-дубликатов;
2.4 Осуществить выборку вычисляемых значений, в результате которой
должна быть сформирована таблица, значения одного столбца в которой получены
путем любых операций (‘+’, ‘–‘, ‘/’, ‘*’) из остальных (результирующий столбец
должен иметь корректное название);
2.5 Выполнить запрос на выборку значений нескольких столбцов одной
таблицы базы данных, с использованием конструкции like;
2.6 Выполнить запрос на выборку значений нескольких столбцов одной
таблицы базы данных, удовлетворяющих одному условию и произвести
упорядочивание по значениям одного столбца;
2.7 Выполнить запрос на выборку значений нескольких столбцов, используя
функции агрегирования данных (SUM, MIN, MAX и т.д.);
2.8 Выполнить запрос на выборку значений нескольких столбцов, используя
функции агрегирования данных (SUM, MIN, MAX и т.д.) используя группировку;
2.9 Выполнить запрос на выборку нескольких столбцов нескольких таблиц
базы данных;
2.10 Выполнить запрос на выборку нескольких столбцов нескольких таблиц
базы данных, используя более 3-х условий;
2.11Выполнить простой вложенный подзапрос к базе данных;
2.12 Выполнить запрос, результатом которого является таблица,
представляющая разность двух исходных таблиц.
